CGDD 4203 - Assignment 1: Roll a Ball (Hyper-Casual)
Unity Version: Unity 6000.0.58f2
Platform Tested: iPhone via Unity Remote 5 (Windows)

Casual Game Design Factors (Hyper-Casual)

Minimal UI and simple objective for fast, short play sessions.

Instant feedback through scoring on target hits.

Quick restart loop after failing.


Game Mechanics:

Player rolls a ball around the arena.

Hitting the target rock increases the score by 1 and the rock respawns at a new location.

The game ends if the ball falls off the platform.

Control UIs (3 Types)

DPad Buttons: On-screen directional buttons for movement.

Tilt Controls: Device tilt moves the ball; includes calibration.

On-screen Joystick: Drag joystick to move the ball.


How to Play:

Choose a control mode using the on-screen control buttons (DPad / Tilt / Joystick).

If using Tilt, press Calibrate while holding the phone comfortably.

Roll into the rock to increase your score.
